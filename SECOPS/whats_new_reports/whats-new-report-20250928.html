What's New Report - 9/28/2025

This report summarizes key updates for Google Security Operations from late September 2025, highlighting new SOAR features, critical API deprecations, new documentation, and informative community posts.

Key Takeaways & Action Items

  • Action Required: The CrowdStrike Detects API will be decommissioned on September 30, 2025. All users must migrate feeds to the new Alerts API to prevent service disruption.
  • New SOAR Features: Chronicle SOAR now includes "Playbook Loops" and "Action Retries" to build more dynamic, resilient, and efficient automation playbooks.
  • New Guides & Content: New community guides are available covering intelligent alert suppression, UEBA-focused detections, UDM parsing deep dives, and using Threat Profiles for proactive defense.
  • Upcoming Events: Hands-on SecOps webinars are scheduled for October and November, covering topics like SOAR Python integrations and Infrastructure as Code.
